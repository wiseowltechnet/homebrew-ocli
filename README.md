# Homebrew OCLI

Homebrew tap for [OCLI](https://github.com/wiseowltechnet/ollama-ocli) - AI coding assistant with self-improvement capabilities.

## Installation

```bash
brew tap wiseowltechnet/ocli
brew install ocli
```

## Usage

```bash
ocli
```

See [main repository](https://github.com/wiseowltechnet/ollama-ocli) for documentation.
